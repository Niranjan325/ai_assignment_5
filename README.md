# ai_assignment_5

# Advanced AI Assignment

## Programs Included

### 1. Minimax Search

Implementation of the Minimax algorithm for adversarial game search.

### 2. Alpha-Beta Pruning

Optimization of Minimax using Alpha-Beta pruning to reduce explored nodes.

### 3. Heuristic Alpha-Beta Search

Combines heuristic evaluation with Alpha-Beta pruning for larger search spaces.

### 4. Monte Carlo Tree Search

Implements random simulations to estimate the best action in a search tree.

### 5. AI Travel Planner

A simple intelligent travel planner recommending tourist places and estimating travel costs using a small knowledge base.

### 6. Knowledge Graphs

A Knowledge Graph is a structured representation of entities and their relationships.

Applications:

* Google Knowledge Graph
* Recommendation Systems
* Semantic Search
* Question Answering

Tools:

* Neo4j
* Protégé
* RDF
* OWL
* Apache Jena
* GraphDB

### 7. Bayesian Networks

Bayesian Networks represent probabilistic relationships among variables.

Example:
Rain → Traffic

Tool Used:

* pgmpy

Inference Method:

* Variable Elimination
